# Visual Language System

The visual system should feel sacred, inward, and timeless.

Primary aesthetic:
- deep charcoal
- black-brown
- muted gold
- low-saturation chakra accents
- subtle gradients
- soft glow
- minimal motion

Avoid:
- neon chakra colors
- cyberpunk effects
- flashy animations
- wellness startup aesthetics
- excessive UI clutter

Animation philosophy:
- breathing
- floating
- aura pulsing
- slow transitions
- gentle fades
- sacred stillness

The app should visually resemble:
- meditation space
- temple silence
- sacred geometry atmosphere
- inner energy flow

Chakra colors must remain elegant and muted.

UI should prioritize:
- calmness
- readability
- immersion
- silence
- focus

Motion should support meditation, not distract from it.
